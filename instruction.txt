PR Reviewer Tool Submission Instructions:

1. Open terminal and navigate to backend folder:
   cd backend

2. Install dependencies:
   npm install

3. Start backend server:
   node server.js

4. Open frontend page:
   frontend/index.html in browser

5. Test workflow:
   - Send a webhook to http://localhost:3000/webhook (Postman or curl)
   - Open frontend to Approve / Reject PR
   - Check reviews.json to see status updates
